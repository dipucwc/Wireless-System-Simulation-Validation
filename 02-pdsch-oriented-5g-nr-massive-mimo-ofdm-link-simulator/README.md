# PDSCH-Oriented 5G NR Massive MIMO-OFDM Link-Level Simulator

This project presents a PDSCH-oriented 5G NR Massive MIMO-OFDM physical-layer link-level simulator for wireless system simulation and validation.

The project includes a technical report, MATLAB implementation, Python reference implementation, simulation result workflow, and Simulink system-level architecture.

## Project Scope

This is a PDSCH-oriented physical-layer simulation project. It focuses on MIMO-OFDM signal processing, channel modeling, DM-RS-based channel estimation, ZF/MMSE equalization, and link-level performance evaluation.

It is not intended to be a full commercial 3GPP conformance test platform.
