# Cross-Verified and Statistically Qualified 5G NR Massive MIMO-OFDM Link-Level Simulator

Design, cross-verification, and statistically qualified performance evaluation of a PDSCH-oriented 5G NR Massive MIMO-OFDM physical-layer link-level simulator.

This publication-oriented project extends the core link-level simulator with verification gates, independent MATLAB–Python cross-verification, Simulink testbench validation, adaptive Monte Carlo stopping, exact confidence intervals, zero-error upper bounds, defect-localization experiments, and reproducibility records.
